# Stochastic Diffusion + Alpha Signal System

## Overview
This project studies stochastic dynamical systems (tamed Navier-Stokes type) and explores their connection to time series modeling and alpha signal generation.

---

## Modules

### 1. SPDE Simulation
- 3D tamed Navier-Stokes inspired stochastic system
- long-term trajectory generation

### 2. Diffusion Feature
- smoothing operator on time series
- extracts latent structure

### 3. Statistical Analysis
- mixing rate estimation
- large deviation (DV) tail risk

### 4. Alpha Construction
- diffusion-based signal
- IC / Rank IC evaluation
- backtesting framework

---

## Key Results
- Stable signal extraction from noisy dynamics
- Positive IC in synthetic experiments
- Evidence of regime-dependent structure

---

## Metrics
- IC
- Rank IC
- PnL
- Mixing decay
- Tail risk (DV rate)
